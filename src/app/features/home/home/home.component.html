2. ALCANCE

Se debe construir un servicio de e-commerce en Spring boot y un framework de frontend que implemente NodeJS y TypeScript (angular, vue, etc.), desde el cual se puedan gestionar: • Productos • Inventarios • Ordenes El producto mínimo viable debe contar con: a) Login b) Creación de usuario c) Gestión de usuarios d) CRUD e) Se deben aprovisionar una serie de reportes: i. Productos activos ii. Top 5 de lo más vendido iii. Top 5 de los clientes frecuentes f) Debe permitir realizar búsqueda de productos por distintos criterios g) Implementación de auditoria
